WP Leads Builder for Any CRM Pro

Leads Builder for Any CRM Pro is a common WordPress plugin that can handle Vtiger, SugarCRM, Zoho CRM and Salesforce to capture or build leads directly from WordPress admin. Now you can generate unlimited forms directly in WordPress admin without any complicated html web forms. Generate web forms via user friendly plugin interface in few simple clicks as short codes. Use the short code which in turn generate as web form in post, page or widget to convert your traffic/visitors as high potential leads to one of your favourite CRM. It supports few third party plugins such as Ninja, Contact form 7 and gravity form plugin.

Feature highlights

  • Generate unlimited forms as short codes
  • Ninja form,contat form 7 and gravity form support added.
  • Duplicate Handling- Choose to skip or update duplicates.
  • Advanced Mandatory Field Form Settings.
  • Assign Captured Lead/Contact To A CRM User.
  • Manage, edit, update, create and delete short codes.
  • Email Notification -Option to get success, failure or both.
  • Google captcha support to avoid spam.
  • Google Captcha version 2 supported.
  • Multiple Language support ( English default , French , Dutch , Russian , Italian ).
  • Capture wordpress User in Lead/Contact module with field mapping option.

Form Builder

Form builder page is used to build the form with the list of enabled fields. Click “Generate Lead Shortcode” or “Generate Contact Shortcode” button to create Form Shortcode. User can enter into form builder page by simply click the edit link in any one of the shortcode presents under manage shortcodes. User can select form type(post or widget), error message submission , success message submission and url redirection specified by post id and can enable the google captcha as shown below.

Select any one of the bulk action from the drop down. The bulk actions are Enable field, disable field, Update order, enable mandatory, disable mandatory and Save display label.

Supported Thirdparty plugins

Leads Builder PRO supports few third party plugins such as Ninja forms, Contact form 7 and Gravity form which enables to make easy form submission. Use the third party shortocode in the post content for form submission. Install the third party plugins via uploads directory.

Note: Use latest version of the plugins.
